# IMC.PYTHON

## Projeto Calculadora de IMC em Python

Este projeto consiste no desenvolvimento de uma Calculadora de IMC (Índice de Massa Corporal) utilizando a linguagem Python. O objetivo principal foi colocar em prática conceitos fundamentais de programação por meio da criação de uma aplicação simples, funcional e interativa.

A aplicação permite que o usuário informe seu peso e altura, realizando automaticamente o cálculo do IMC através da fórmula:

IMC = peso / altura²

Após o cálculo, o sistema apresenta o valor do IMC e também a classificação correspondente, seguindo os padrões utilizados pela Organização Mundial da Saúde (OMS), como:

* Abaixo do peso
* Peso normal
* Sobrepeso
* Obesidade grau I
* Obesidade grau II
* Obesidade grau III

Durante o desenvolvimento do projeto, foram aplicados diversos conceitos importantes da programação em Python, entre eles:

* Entrada e saída de dados com `input()` e `print()`
* Uso de variáveis para armazenamento de informações
* Operações matemáticas
* Estruturas condicionais 
* Conversão de tipos de dados
* Organização e legibilidade do código

Além do cálculo em si, o projeto contribuiu para o desenvolvimento do raciocínio lógico e da capacidade de criar soluções práticas utilizando programação.

A proposta do sistema é simples, mas muito importante para quem está iniciando no desenvolvimento de software, pois ajuda a compreender a estrutura básica de um programa e a interação com o usuário.

Esse projeto também pode servir como base para futuras melhorias, como:

* Criação de interface gráfica
* Armazenamento de dados em banco de dados
* Histórico de cálculos
* Integração com aplicativos de saúde
* Versão web ou mobile

O desenvolvimento desta atividade foi fundamental para reforçar conhecimentos básicos em Python e entender melhor como transformar lógica em aplicações funcionais.
